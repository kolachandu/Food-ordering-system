# Food Ordering System

A simple Java console application for placing food orders.

## Features

- View the food menu
- Add items to the cart
- Remove items from the cart
- See subtotal, GST, delivery fee, and total
- Checkout with customer details
- Print an order confirmation

## Run

```powershell
javac -d out src/Main.java
java -cp out Main
```
